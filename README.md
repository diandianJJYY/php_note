#7月4日 总结
### 运算符

------
位运算符 & （按位与） 和 逻辑运算符 &&（逻辑与）：
```php
<?php
$a = (2 & 5);
echo $a; // 0  ( 0 = 0000) = ( 2 = 0010) & ( 5 = 0101)
$b = (2 && 5);
var_dump($b); // bool(true) （真 && 真）= 真

?>
```
### 流程控制：
>for(表达式1，表达式2，表达式3）{}
> * 表达式1：在循环开始前无条件求值（并执行）一次。
> * 表达式2：在每次循环开始前求值。如果值为 TRUE，则继续循环，执行嵌套的循环语句。如果值为 FALSE，则终止循环。
> * 表达式3：在每次循环之后被求值（并执行）。
